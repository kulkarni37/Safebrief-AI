# 🛡️ SafeBrief AI
### AI-Powered Industrial Safety Intelligence Platform

SafeBrief AI is an intelligent industrial safety platform that leverages Artificial Intelligence, Large Language Models (LLMs), Machine Learning, and Retrieval-Augmented Generation (RAG) techniques to analyze incident reports, predict risk levels, provide actionable recommendations, and assist users through an AI-powered safety assistant.

---

## 🚀 Features

✅ AI Incident Report Summarization

✅ PDF Incident Report Analysis

✅ Image-Based Safety Analysis

✅ Video Incident Analysis

✅ ML-Based Risk Prediction

✅ AI Safety Chatbot Assistant

✅ Similar Incident Memory Retrieval

✅ Automated Recommendation Generation

✅ Downloadable AI Generated PDF Reports

✅ Interactive Dashboard & Visual Analytics
